# ❤️ Brisa & Vini - Nossa História

Este é um projeto pessoal e romântico desenvolvido como uma **Landing Page** especial para celebrar o meu relacionamento com a Brisa. O site funciona como um memorial interativo, repleto de momentos, música e tecnologia.

## 🚀 Funcionalidades

* **Contador de Tempo Real:** Exibe meses, dias, horas, minutos e segundos desde o início da nossa jornada (30/01/2026).
* **Chuva de Corações:** Efeito visual dinâmico com corações flutuantes que preenchem a tela.
* **Galeria de Momentos:** Carrossel interativo com fotos e vídeos dos nossos melhores dias.
* **Trilha Sonora:** Player de música integrado para tocar aquela canção que me faz lembrar dela.
* **Mapa Especial:** Localização exata de onde nossa história começou, com visual personalizado.
* **Design Responsivo:** Totalmente otimizado para ser visualizado tanto no computador quanto no celular.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estrutura semântica da página.
* **CSS3:** Estilização avançada com animações, filtros e efeitos de desfoque (Glassmorphism).
* **JavaScript (Vanilla):** Lógica do contador, carrossel de mídia e efeitos de chuva de corações.
* **Google Maps API:** Incorporação do mapa do nosso lugar especial.

## 📸 Como visualizar
Para abrir o projeto, basta baixar os arquivos e abrir o `index.html` em qualquer navegador ou acessar através do **GitHub Pages**.
